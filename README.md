# WinNetshDesktop

为了方便的管理 windows netsh 代理工作，我们开发此软件。

- 接口代理
- 防火墙规则

两者配合能够很好的管理

例如：如果我们有一个 `http://localhost:8080` 的服务, 我们需要外部能够访问到，这里我们

- 配置代理
- 添加防火墙入栈规则（如果没有）

配置好之后就能很方便的管理，

## 使用示例

- Stable Diffusion web 服务 只提供一个本地端端口 `http://127.0.0.1:7860/`，如果我想在手机上使用就可以很多好的处理。
